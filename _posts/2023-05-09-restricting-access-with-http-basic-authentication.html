To restrict access to your website using HTTP Basic Authentication with nginx, you can follow these steps:

  1. Create a password file: This file contains the usernames and passwords that are allowed to access your website.Verify thatapache2-utils(Debian, Ubuntu) orhttpd-tools(RHEL/CentOS/Oracle Linux) is installed.. For example, to create a user named “john” with the password “password”, you would run the following command: htpasswd -c /path/to/.htpasswd john

  2. Configure nginx: You need to configure nginx to use the .htpasswd file for authentication. You can do this by adding the following lines to your nginx configuration file:

location / {

    auth_basic "Restricted Content";

    auth_basic_user_file /path/to/.htpasswd;

}

  1. Restart nginx: After making changes to your nginx configuration file, you need to restart nginx for the changes to take effect.

That’s it! Now when someone tries to access your website, they will be prompted for a username and password.
