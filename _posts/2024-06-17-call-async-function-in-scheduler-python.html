  1. Define download_and_parse_files as an asynchronous function.

  2. Use a separate function to handle the invocation of the async function within the job.

Here’s a full example:

import asyncio
from apscheduler.schedulers.background import BackgroundScheduler

# Define your asynchronous function
async def download_and_parse_files():
    # Your async code here
    await asyncio.sleep(1)
    print("Files downloaded and parsed")

# Define a wrapper function to run the async function
def run_async_job():
    asyncio.run(download_and_parse_files())

# Create a scheduler instance
scheduler = BackgroundScheduler()

# Schedule the job
scheduler.add_job(run_async_job, 'interval', seconds=10)

# Start the scheduler
scheduler.start()

# Keep the script running
try:
    while True:
        pass
except (KeyboardInterrupt, SystemExit):
    scheduler.shutdown()

Explanation:

  1. Asynchronous Function: download_and_parse_files is defined as an async function using async def.

  2. Wrapper Function: run_async_job is a synchronous function that calls asyncio.run(download_and_parse_files()).

  3. Scheduler Setup: scheduler.add_job(run_async_job, 'interval', seconds=10) schedules the run_async_job function to run every 10 seconds.

By separating the async function from its invocation, this approach ensures compatibility with APScheduler, which expects synchronous callable jobs.

If we want to call the async function directly in scheduler, then use this line

scheduler.add_job(lambda:asyncio.run(download_and_parse_files()), 'interval', seconds=10)
