I am new to go, and I found myself somehow faces the issue of package path and importing issue. I would like to share how I solve it.+

There are few easy steps:

1, find your gopath: go env GOPATH, it is normal at ~/go

2, in the ~/go/src, for GitHub, create folder github.com/xiaobojiang/

3, create a repo, eg go-playground in the git, and pull, in this folder

4, init the module: go mod init github.com/xiaobojiang/go-playground

5, create a module with below directory tree

../serverlissajous

├── lissajous

│   └── lissajous.go

└── serverlissajous.go

Serverlissajous will use the package lissajous, best practice is that package name shall be same as folder name

Lissajous is a module, snippet of code
