Replacing the default ubuntu source to domestic source (tsinghua tuna), this greatly improves the apt update and download speed.

Since I use this a lot, I would like to share the steps for later reference.

Manual replacement

Confuguration file of Ubuntu source is /etc/apt/sources.list.

Before you make the changes, please make a backup by copy this file with a new name.

Below lines of configuration is different for various ubuntu release, please find yours in below reference link.

Tsinghua tuna's source list shall be 

# by default, comment out source code link to improve speed, uncomment if required
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy main restricted universe multiverse
# deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy-updates main restricted universe multiverse
# deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy-updates main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy-backports main restricted universe multiverse
# deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy-backports main restricted universe multiverse
deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy-security main restricted universe multiverse
# deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy-security main restricted universe multiverse

# re-release source, do not recommand
# deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy-proposed main restricted universe multiverse
# deb-src https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy-proposed main restricted universe multiverse

Command replacement

Execute below command to replace:

sudo sed -i "s@http://.*archive.ubuntu.com@https://mirrors.tuna.tsinghua.edu.cn@g" /etc/apt/sources.list
sudo sed -i "s@http://.*security.ubuntu.com@https://mirrors.tuna.tsinghua.edu.cn@g" /etc/apt/sources.list

That's it.

execute below commands to use this new source and update, enjoy!

$ sudo apt update 
$ sudo apt upgrade

Reference: https://mirror.tuna.tsinghua.edu.cn/help/ubuntu/
