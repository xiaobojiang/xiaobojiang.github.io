In today's digital age, the proliferation of Artificial Intelligence (AI) has revolutionized how we interact with technology, offering convenience and efficiency across various platforms. One significant area where AI is increasingly integrated is in chat interfaces, where users engage in conversations ranging from customer support queries to personal assistance. However, amidst the benefits lie critical concerns regarding the protection of privacy data.

I found this tool - Presidio from Microsoft which can help this.

Presidio can help identify sensitive/PII data in un/structured text.

Presidio (Origin from Latin praesidium ‘protection, garrison’) helps to ensure sensitive data is properly managed and governed. It provides fast identification and anonymization modules for private entities in text such as credit card numbers, names, locations, social security numbers, bitcoin wallets, US phone numbers, financial data and more.

Git link: https://github.com/microsoft/presidio

Let's try this.

Installation:

$ pip install presidio_analyzer

$ pip install presidio_anonymizer

$ python -m spacy download en_core_web_lg

It utlizes spacy to download model en_core_web_lg (around 500+ MB) and use it.

Sample code

from presidio_analyzer import AnalyzerEngine

from presidio_anonymizer import AnonymizerEngine

text="My phone number is 212-555-5555"

#Set up the engine, loads the NLP module (spaCy model by default) and other PII recognizers

analyzer = AnalyzerEngine()

Call analyzer to get results

results = analyzer.analyze(text=text,

                          entities=["PHONE_NUMBER"],

                          language='en')

print(results)

Analyzer results are passed to the AnonymizerEngine for anonymization

anonymizer = AnonymizerEngine()

anonymized_text = anonymizer.anonymize(text=text,analyzer_results=results)

print(anonymized_text)
