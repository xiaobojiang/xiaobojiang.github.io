LLM (large language model) can help people do many tasks: information fetching, text processing, Q&A. There are powerful and popular models avaiable on the internet, eg chatgpt, claude, bard. They are big in parameters and close source. What if you can deploy some open source model locally and free to use, even you are in a flight, sounds interesting!

The open source GPT4All(https://gpt4all.io/index.html) and MLC(https://llm.mlc.ai/), and closed source LM Studio (https://lmstudio.ai/)(which has a very nice user interface) are making it easier than ever to run models locally.

Let's pick GPT4All to start, this is a github project with high stars (55K+ at 2023.12).

We can download the package for different OS at https://gpt4all.io/index.html.

Install it, and now, you can select a model to start.

There are various model can be installed locally, and each has different size and RAM requirement.

I picked the first recommendation, mistral-7b-openorca.

Try a sample question: "how many oceans on the Earth, give me a list."

It works pretty good.

User can download multiple models and switch between them.

Regarding the speed, it works on my Macbook pro with M1 pro chipsets. Average output speed is around 35 tokens/second, around 25 words per second. Really not bad.

Keep learning, and enjoy!
