If you would like to print down the certificate content, without transfer '\n' into linebreak. Normally, this kind of raw content is required in uploading cert in the api calls.

You can try this fuction: repr()

A snipet of example code:

with open('./ca/cacert.pem','r') as f:
    lines = f.readlines()

print(repr(''.join(lines)))
