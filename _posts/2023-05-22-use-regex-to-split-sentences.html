The regex (?<=[.!?]) is called a positive lookbehind. It is used to match a pattern that is preceded by another pattern. In this case, it matches any position that is immediately preceded by a period, exclamation mark or question mark. This is useful when you want to match a sentence that ends with one of these punctuation marks.

For example, the regex (?<=[.!?])\s+ matches any whitespace character that follows one of these punctuation marks and can be used to split a string into sentences.

Sample Code:

import re
>>> text = " T?e  qu!ck ' brown 1 fox! jumps-.ver. the 'lazy' doG?  !"
>>> re.split('(?<=[.!?]) +',text)
[" T?e  qu!ck ' brown 1 fox!", 'jumps-.ver.', "the 'lazy' doG?", '!']
