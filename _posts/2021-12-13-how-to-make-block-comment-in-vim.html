Vim is a great tool, for me, it especially useful when I am changing files in Linux command line without GUI.

One of daily usage is to comment and uncomment lines in Vim. I searched and found few nice ways to do this and would like to share in this post. In general, there are two ways: 1, original 2, use plugin

Original means that we are using some original grammar that vim supports, no further plugin installation is required.

Original Way

Take one example, I have one file called test.py, I want to comment all 3 lines.

 
print ("hello world")
print ("hi")

There are few steps:

1, go to command mode, press ‘v’ to enter visual select mode, press arrows to select few lines that we want to commen

2, press :s/^/#, this make replace the starting of line with character with Python comment char ‘#’. Press enter

3, you have got these lines commented

To uncomment these line, procedure is similar, only difference is in step 2, replace the command string with :s/^#/

For C, the comment is “//”, since ‘/’ is a special character in Vim, we need to use ‘\’ to escape.

So the comment command in step 2 is :s/^/\/\/

And the uncomment command in step 3 is :s/^\/\//

Plugin way

I recommend to use nerdcommenter. We can use Vundle to manage vim plugins and install it.

Follow the guide in https://github.com/preservim/nerdcommenter.

Installation:

  • Add Plugin 'preservim/nerdcommenter' to your vimrc file.

  • Reload your vimrc or restart

  • Run :BundleInstall

Post Installation

  • Make sure that you have filetype plugins enabled, as the plugin makes use of |commentstring| where possible (which is usually set in a filetype plugin). See |filetype-plugin-on| for details, but [count]<leader>cu |NERDCommenterUncomment|

Uncomments the selected line(s).

filetype plugin on

How to comment lines with nerdcommenter:

1, find your leader key

Type :echo mapleader in the vim command mode.

And the leader key will pop up, for MacOS, it is ‘,’.

2, in visual mode, select few lines

  • [count]<leader>cc 

Comment out the current line or text selected in visual mode.

For my system, for the selected lines, enter ,cc can comment all lines

  • [count]<leader>cu 

Uncomments the selected line(s).

For in my system, for the selected lines, enter ,cu can uncomment all lines

Similarly, 2,cc  2,cu can comment/uncomment 2 lines below cursor.

Using plugin is simpler, can it can adapt different languages.

Conclusion

In this post, two ways of comment lines of code are explained.

If you daily use Vim on one or two computers, plugin is a nice thing. If you are editing on a new server, using original way is more light weighted.

In the end, use one in your preference.
