Problems to solve:

There is a service running in my Ubuntu, I want to check periodically whether it is still running, and take action accordingly.

Solution:

1, check service running status

systemctl is-active --quiet <service-name>

If the service is running, it will exit with code zero.

2, append following action

in shell, for the conditional code, && and || tokens allow us to connect our commands with the help of their exit status.
The && token chains the commands executing what is on the right if and only if the instruction on the left had an exit status of zero.
And the || token, chain the commands executing what is on the right if and only if the instruction on the left had an exit status different from zero.

so, the example can be:

systemctl is-active --quiet sshd && echo "sshd is running" || echo "sshd is NOT run

3, equiped with cron

cron can help you add a periodically job.

This helps me to check on every hour, if the frpc service is not running, restart it.

 # m h dom mon dow command 
0 * * * * systemctl is-active --quiet frpc || systemctl restart frpc

That's it.
