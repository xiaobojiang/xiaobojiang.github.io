There are cases that we want to ssh to remote server, and execute some command, without really logon to the machine. For example, I want to use CI/CD tool to deploy something on remote machine.

It is easy, standard command is:

$ ssh host:ip 'command to execute'

Simple example:

$ ssh host ‘ls’

‘ls’ is the command you want to execute on the machine

For remote update

Consider this Jenkins build script, "build.sh"

export testenv=helloworld
envsubst < test.sh | sh

Deploy script, test.sh

ssh gateway_qa_shanghai 'echo ${testenv}’

Then try build.sh

$sh build.sh

[13:52:43]

helloworld

It will output hello world from the env we set in the script.

Replace this with docker run alpine:${tag} or similar command, then it can do image update on the remote machine.
