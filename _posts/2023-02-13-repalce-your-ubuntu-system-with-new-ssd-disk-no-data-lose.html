I have one HP Z820 workstation, which is kind of old, but still good in use. It has one 512G HDD disk, in nowadays, is kind of slow in system startup and IO throughput. So I purchase 1 samsung 1T SSD disk, tried to totally replace the HDD disk, and without reisntalling the system and keep data. Below is how I do it.

Preparation:

1, install your new SSD disk into the PC

2, prepare a Ubuntu installation USB disk

Steps:

System Boot:

  • boot your system from USB disk (set the bios start up sequence, move USB priority as highest)

Move data from old disk to new:

  • start gparted from ubuntu menu

  • Copy data from old disk:

1, Select disk from upper corner

2, copy the partition that we want to move

3, Select the target device and paste the copied partition. If the device is in the first use, you need to create a new partition table before the paste.

  • Set the flag: Boot and others

1, check the flag on existing boot disk, and set the same in the new disk

  • Apply the changes

  • Shut down the Ubuntu

  • Remove the old disk and flash disk from PC

  • Restart the PC, optionally set the new disk as first boot option

Now, your new SSD disk shall working, and system will boot with all your data present.

We can connect back the HDD disk, and then erase the data, use it as backup disk.
