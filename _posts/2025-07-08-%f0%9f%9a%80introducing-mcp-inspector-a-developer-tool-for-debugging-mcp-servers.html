TheMCP Inspectoris a powerful, browser-based tool for testing and debugging MCP (Model Context Protocol) servers. Whether your server runs over stdio, SSE, or streamable-http, this tool provides an interactive UI and a CLI mode for full visibility and control.

🧪 Quick Start (UI Mode)

Launch the Inspector UI in seconds:

npx @modelcontextprotocol/inspector

Then open the inspector with token pre-filled url:

Then you can select one of three transport type and connect for testing: Stdio, SSE (to be deprecated), Streamable https.

Take stdio io as example, we can connect to a server via GUI configuration:

Put command eg: uv inside, and then arguments (uv command arguments) : --directory /Users/xiaobo/MyGit/mcp_playground/weather run weather.py (this is the example from MCP official site)

Then, click connect, it will trigger initialization.

Then if we go to tool tab, click button "List Tools", now this tool presents two tools in this server: "get_alerts" and "get_forecast"

Great!

 Follow similar way, we can connect to Streamable http server, be care to put the link and set the auth setting (eg bearer token).

Another way to connect a MCP server is via command line.

We can also the same stdio server by configuring below command:

$ npx @modelcontextprotocol/inspector uv --directory /Users/xiaobo/MyGit/mcp_playground/weather run weather.py

This could auto fill the stdio parameters in the url.

Finally, we can select a tool, fill in the parameter and run the tool.

Cool! As in the screen, I can get the weather alerts from California.

With this we can quickly testing the MCP server and move on to integrate into your AI agent!

Hope you enjoy this!
