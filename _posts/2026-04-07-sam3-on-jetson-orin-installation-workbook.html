This document is a replicable, end-to-end guide to install and run SAM3 (Segment Anything Model 3) on NVIDIA Jetson AGX Orin with JetPack 6.2 / CUDA 12.6 / Ubuntu 22.04.

SAM3 is really a powerful model that can run on Jetson for object segmentation in image or video, it can work as out of box AI vision with rather good accuracy, but without further transfer learning on certain models (eg, Yolo)

It captures all pitfalls, fixes, and dependencies discovered during real deployment.

Overall for uv setup

If you want to totally run SAM3 with UV in venv. follow below points during full installation guide

uv python install 3.10
uv venv --python 3.10
source .venv/bin/activate
uv init
uv add huggingface_hub

Using uv pip install to add pytorch package

uv pip install --index-url https://pypi.jetson-ai-lab.io/jp6/cu126 torch torchvision --force-reinstall

There will be few missing packages need to resolve for pytorch and sam3 source build, referred in SAM3_JETSON, details in below sections:

  • cudss

  • decord

add local decord build with uv add local

1. System Verification

1.1 Check JetPack / L4T Version

head -n 3 /etc/nv_tegra_release

Expected example:

# R36 (release), REVISION: 4.7

This corresponds to JetPack 6.2.x (L4T 36.4.x).

1.2 Check CUDA Version

nvcc --version   #optional
nvidia-smi

Expected:

Cuda compilation tools, release 12.6

2. Install Correct PyTorch (Jetson-Compatible)

⚠️ Do NOT use pip install torch from PyPI
Jetson requires ARM64 CUDA-enabled builds.

2.1 Remove Existing PyTorch

python3 -m pip uninstall -y torch torchvision torchaudio

2.2 Install Jetson-Specific PyTorch

python3 -m pip install --index-url https://pypi.jetson-ai-lab.io/jp6/cu126     torch torchvision --force-reinstall

This installs PyTorch built for:

  • ARM64

  • CUDA 12.6

  • JetPack 6.x

2.3 Fix Missing cuDSS Library (Critical)

If you see:

ImportError: libcudss.so.0: cannot open shared object file

Install cuDSS:

wget https://developer.download.nvidia.com/compute/cudss/0.6.0/local_installers/cudss-local-tegra-repo-ubuntu2204-0.6.0_0.6.0-1_arm64.deb
sudo dpkg -i cudss-local-tegra-repo-ubuntu2204-0.6.0_0.6.0-1_arm64.deb
sudo cp /var/cudss-local-tegra-repo-ubuntu2204-0.6.0/cudss-* -t /usr/share/keyrings/
sudo apt-get update
sudo apt-get -y install cudss

Verify:

ls /usr/local/cuda/lib64 | grep cudss

2.4 Verify PyTorch CUDA Works

python3 - << 'EOF'
import torch
print("Torch:", torch.__version__)
print("CUDA available:", torch.cuda.is_available())
x = torch.rand(1, device="cuda")
print("CUDA tensor:", x)
EOF

3. Install SAM3

3.1 (Optional) Create Virtual Environment

python3 -m venv ~/.venvs/sam3
source ~/.venvs/sam3/bin/activate

3.2 Install SAM3

built from source, instead of uv add or pip install, because it will install latest cuda and torch, which will break packages installed already on Jetson. (Note: Jetson requires its unique pytorch compiled, you can install it from special repo: https://pypi.jetson-ai-lab.io/jp6/cu126

git clone https://github.com/facebookresearch/sam3.git
cd sam3

# activate your uv env
source ~/sam3-vision-app/.venv/bin/activate

# install in editable mode (uv version)
uv pip install -e .

⚠️ SAM3 does not declare all dependencies. Missing imports are expected.

4. Install Missing Python Dependencies?

There will be few packages missing, just uv add one by one.

pip install einops psutil numpy opencv-python

5. Install decord (ARM64 – Build from Source)

decord has no official ARM64 wheel. so we need to built from source, and add it properly to uv dependencies (refer to overall note)

5.1 Install System Dependencies

sudo apt-get update
sudo apt-get install -y   git cmake build-essential   ffmpeg libavcodec-dev libavformat-dev libavutil-dev

5.2 Build decord

git clone https://github.com/dmlc/decord.git
cd decord
git submodule update --init --recursive

mkdir build && cd build
cmake .. -DUSE_CUDA=0 -DCMAKE_BUILD_TYPE=Release
make -j$(nproc)
sudo make install

cd ../python
python3 -m pip install .
#for uv
uv pip install .

Verify:

python3 - << 'EOF'
from decord import VideoReader, cpu
print("Decord OK")
EOF

6. Run SAM3

Example test:

from sam3.model_builder import build_sam3_image_model

model = build_sam3_image_model()
print("SAM3 model loaded successfully")
