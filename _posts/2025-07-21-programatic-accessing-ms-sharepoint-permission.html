Microsoft sharepoint is an Enterprise software, experts users using this to store documents and share with others.

In the new AI era, one of the topic is to build AI agent that can make best use of these documents and assist people in daily work, for example, an RAG based knowledge assistant.

One of the task to connect our AI framework with this sharepoint, thus it can get latest document update. It has many advantages:

  • SharePoint is part of Microsoft 365 and integrates with Azure AD, providing fine-grained access control.

  • You can define permissions per file, folder, or user group.

  • Supports audit logs and compliance (e.g., for ISO, HIPAA).

How to enable this? follow this tutorial

1, create an Azure managed application, and take client id/secret, applicaiton name, and tenent id from it.

make sure some API permission are properly set here

2, go to https://developer.microsoft.com/en-us/graph/graph-explorer

2.1 login yourself

3, find the site id, one way is to follow the site, and call this api

GET https://graph.microsoft.com/v1.0/me/followedSites

Site id is the string of key "id"

4, call API to grant access (you must be sharepoint admin)

POST

https://graph.microsoft.com/v1.0/sites/{siteid}/permissions

body:

{

  "roles": ["read"],

  "grantedToIdentities": [{

  "application": {

  "id": "7c...",

  "displayName": "XXX"

}

You shall get something like:

code 201

Done

For step 4, there is also GUI way of working, check this document's step 2
