The find module in Ansible is used to search for files, directories, or links based on specific criteria such as age, pattern, or regular expression. One of the features of this module is that you can use the patterns parameter to select files that match a shell-style glob. For example, you can use patterns: '*.txt' to find all the text files in a given path.

Some additional information that might be helpful:

We can use the recurse parameter to enable or disable recursive search.
We can use the register keyword to store the output of the find module in a variable for later use.
We can use the file_type parameter to filter by file type, such as file, directory, link, socket, etc.

Below is the sample code that:

1, find all the tar files in the specified path, and register the result in "find_result"

2, read the file json object list in parameter "files", and use map method to get paths as list

3, now use the list in loop to operate on them.

---
# tasks file for application
- name: Get the current applications running
  find:
    paths: /path/to/app
    file_type: file
    Patterns: “*.tar"
  register: find_result

- name: Print find output

    var: find_result.files

- name: Print find output
  debug:
      msg: "{{ item }}"
  with_items: "{{ find_result.files|map(attribute='path')|list }}"
