It is a common security practice to disable password authentication for ssh service.

There is no module in Ansible to do that, here I would like to share my script, use it if you have need

Create a role with following structure, role name "disable_ssh_password":

disable_ssh_password.yml
├── files
├── handlers
│ └── main.yml
├── tasks
│ └── main.yml
├── templates
└── vars

content of tasks/main.yml:

---

# tasks file for sshd-config

---
- name: sshd PasswordAuthentication set to '{{ sshd_PasswordAuthentication }}'
  lineinfile:
    dest: /etc/ssh/sshd_config
    regexp: '^#?PasswordAuthentication'
    line: 'PasswordAuthentication {{ sshd_PasswordAuthentication }}'
    validate: 'sshd -t -f %s'
  notify: restart ssh service
  when: sshd_PasswordAuthentication is defined

What it does:

  • lineinfile module is to find the target password authenticaion setting line and replace it with desired value

  • target line starts "#PasswordAuthentication" or "PasswordAuthentication", with regexp string to find

  • replaced line is templated with desired value like "PasswordAuthentication no"

  • validate command is to validate sshd configuration file

  • notify calls below ansible handler to create a task to restart sshd service to make new configuration effective.

handlers/main.yml

this file create the handler for notify command to call.

---
# handlers file for sshd-config
- name: restart ssh service
  service:
    name: sshd
    state: restarted

Note: validate method and lineinfile command requires sudo access in Ubuntu, otherwise it will complain with no hostkey string error.

Add become: yes in the playbook if applicable

Example Playbook

create a playbook with name disable_ssh_passsword_playbook.yml, content like

- hosts: iotcolon
  roles:
    - role: disable_ssh_password
      sshd_PasswordAuthentication: 'no'
  become: yes

Alright, run the book like

$ ansible-playbook -i /path/to/env disable_ssh_password_playbook.yml

the sshd configuration is updated, and target host cannot be ssh anymore with password authentication.
