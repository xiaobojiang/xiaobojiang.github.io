When working with AWS EC2 or Lightsail virtual machines (VMs), knowing your public IP address is essential for managing network access or setting up services. There are various ways to get the public IP address of your instance directly from the terminal without relying on third-party websites.

Below, I'll walk through some quick and efficient methods to retrieve your public IP address on AWS EC2 or Lightsail.

Step 1: Connect to Your AWS Instance via SSH

Before fetching the public IP address, you need to SSH into your AWS EC2 or Lightsail instance. Use the following command:

$ ssh ec2-user@<instance_name>

Replace <instance_name> with the name or IP address of your instance.

Step 2: Use dig to Find Your Public IP

One of the fastest methods to find the public IP address of your instance is to use the dig command with DNS resolvers that return your public-facing IP.

Method 1: Using OpenDNS Resolver

Run the following dig command to get your public IP via OpenDNS:

$ dig +short myip.opendns.com @resolver1.opendns.com

This command queries OpenDNS's resolver, which responds with the public IP of your instance. It's a simple and reliable method.

Method 2: Using Google DNS Resolver

Alternatively, you can use Google’s DNS resolver with this command:

$ dig TXT +short o-o.myaddr.l.google.com @ns1.google.com

The response will include your public IP address, typically in a text format

Step 3: Use curl to Retrieve IP

Another convenient option to check your public IP is by using a simple HTTP request with curl. AWS provides an endpoint that returns your public IP address:

$ curl http://checkip.amazonaws.com

This command will output your public IP address as plain text.

Why These Methods are Preferred

  • Speed: These commands directly query DNS servers or an AWS endpoint, making them faster and more reliable than using external websites.

  • Security: Since you’re not relying on third-party services, the commands are more secure.

  • Simplicity: No additional software or tools are needed beyond basic utilities like dig or curl, which are often pre-installed on most Linux distributions.

Summary of Commands

Conclusion

Knowing how to quickly find the public IP address of your AWS EC2 or Lightsail instance can save you time and make managing your instance more efficient. Whether you use dig or curl, these methods are reliable and don’t require third-party services, ensuring both speed and security.
