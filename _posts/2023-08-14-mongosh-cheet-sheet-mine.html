1, create user and authenticate user with access to one db

docker exec mongo mongosh -u {{ MONGO_ROOT_USERNAME }} \

--eval 'use admin' \

--eval 'db.createUser({user: "{{ MONGO_USER }}", pwd: "{{ MONGO_PWD }}", roles: [ { role: "readWrite", db: "{{ MONGO_DATABASE }}" } ]})' \

-p '{{ MONGO_ROOT_PASSWORD }}'

2, login to mongosh

docker exec mongo mongsh -u {{ MONGO_ROOT_USERNAME }} -p '{{ MONGO_ROOT_PASSWORD }}

In the mongosh.

Some useful commands:

1, use [dbname]

2, db.[collection name].find() show all

More methods are documented here: https://www.mongodb.com/docs/v6.0/reference/method/

3, find filter

db.material_translation.find({"language":"Dutch”})

Found more here https://www.mongodb.com/docs/v6.0/reference/method/db.collection.find/#mongodb-method-db.collection.find

4, drop

db.material_translation.drop()

Drop all data in the collection
