yt-dlp is a command-line program that lets you easily download videos and audio from more than a thousand websites. See the list of supported sites (https://github.com/yt-dlp/yt-dlp/blob/master/supportedsites.md)

If you have python, installation is easy:

$ pip install yt-dlp

Then use it in your command line to download the video:

$ yt-dlp URL

extract the audio-only for downloads:

$ yt-dlp -x -f bestaudio URL

For example, extra the audio from youtube, this can be the input for speech-text and later AI analysis

yt-dlp -x -f bestaudio https://www.youtube.com/shorts/dol2GgVE5Vs

Also works for bilibili:

yt-dlp -x -f bestaudio https://www.bilibili.com/video/BV1oe411U7vi/\?spm_id_from\=333.1007.tianma.1-1-1.click\&vd_source\=a83709781a3d15f1317ea9a28285679d
