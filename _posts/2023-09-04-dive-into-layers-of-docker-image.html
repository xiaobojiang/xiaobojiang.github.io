Dive is a very handy tool to inspect docker image. it is "A tool for exploring a docker image, layer contents, and discovering ways to shrink the size of your Docker/OCI image."

I use this tool to review the changes I wrote in the dockerfile, and make sure everything is executed correctly.

Furthermore, we can use this tool to analyze the files and packages in the image.

It is hosted in github, https://github.com/wagoodman/dive.

Prepare a shell script, name is dive.sh

echo $1
docker run --rm -it \
-v /var/run/docker.sock:/var/run/docker.sock \
quay.io/wagoodman/dive:latest $1

Use dive again a image by following command:

$ sh dive.sh <image url or id>

Now we shall see UI similar to below image.

Upper left pannel is the layers of this image, including all steps we requested in the dockerfile.

Lower left pannel is the details of the image.

Right pannel is details of layer content, we can know which files are added or updated by exploring in the panel view.

In the right panel:

How to view changes: up and down in layer

How to see files changed: tab to “Current Layer details”

In current Layer details: shift + space to unfold and explore changes.
