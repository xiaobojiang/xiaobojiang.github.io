How to type the check mark (✓) character.

Here are different ways depending on your system:

✅ 1. Copy–Paste Directly

You can just copy these:

  • ✓ (U+2713)

  • ✔ (U+2714, bold version)

  • ✅ (emoji version)

✅ 2. Windows

  • Press Alt key + code on numeric keypad:

    • Alt + 10003 → ✓

    • Alt + 10004 → ✔

  • Or open Character Map (charmap) and search “check”.

✅ 3. macOS

  • Press Control + Command + Space to open the emoji & symbol viewer.

  • Search “check” or “tick” → select ✓ or ✔.

✅ 4. Microsoft Word / Office

  • Type 2713, then press Alt + X → ✓

  • Type 2714, then press Alt + X → ✔

✅ 5. HTML / Markdown

  • ✓ → ✓

  • ✔ → ✔

⚡ Tip: If you only need it casually, the fastest way is to copy and paste ✓.
