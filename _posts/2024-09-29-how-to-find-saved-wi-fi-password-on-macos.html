I migrated to MacPro from windows years ago, in windows, finding password of connected WiFi is easy (via view setting of this WiFi SSID). It is not so easy in MacOS, well, security and privacy is well protected, and you may need few more steps to get this.

To view the saved Wi-Fi password on macOS, you can use the Keychain Access app or the Terminal. Here’s how to do it with both methods:

Method 1: Using Keychain Access

1. Open Keychain Access from the Applications > Utilities folder, select "System" on the left-hand navigation panel.

2. In the search bar, type the name of the Wi-Fi network.

3. Double-click the network from the list.

4. In the window that appears, check the box labeled Show Password.

5. You will be prompted to enter your administrator username and password.

6. Once authenticated, the Wi-Fi password will be displayed in the password field.

Method 2: Using Terminal

1. Open Terminal from Applications > Utilities.

2. Type the following command and replace NETWORK_NAME with the name of your Wi-Fi network:

security find-generic-password -a "NETWORK_NAME" -gw

-a "NETWORK_NAME"

• -a (Account): This option specifies the account name (in this case, the Wi-Fi network name or SSID).

• "NETWORK_NAME" is the name of the Wi-Fi network whose password you are trying to retrieve. By specifying this, the command knows which specific entry in the keychain you are interested in.

3. You will be prompted to enter your administrator password.

4. After authenticating, the Wi-Fi password will be displayed in the terminal.
