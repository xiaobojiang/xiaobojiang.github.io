If you want to run a Panel app outside of a Jupyter notebook, you can use the panel serve command. Let's modify the example to include instructions for running the app without Jupyter:

import panel as pn

# Define a function that returns a Panel object
def greeting(name):
    return pn.pane.Markdown(f"## Hello, {name}!")

# Create an interactive widget
name_input = pn.widgets.TextInput(name="Enter your name:", placeholder="Type here")

# Combine the widget and function to create a reactive Panel app
app = pn.Column(name_input, greeting)

# Display the app in a standalone mode (without Jupyter)
if __name__ == "__main__":
    app.servable()

With this modification, you can run your app as a standalone application using the following command in the terminal:

$panel serve app.py

This command will start a local server, and you can view your app by navigating to http://localhost:5006 in your web browser.

Now, the if __name__ == "__main__": block ensures that the app.servable() line is only executed when the script is run directly, not when it's imported as a module in another script.

Feel free to run this script from the terminal, and you should see your Panel app running as a standalone web application.
