ESP32, and its predecessor ESP8266, are famous WiFi SoC for IoT application, which are created and developed by Espressif Systems. It contains WiFi and Bluetooth wireless abilities and stacks, there are also various examples in Github, all these make it very suitable for IoT playground.

And there are also many commercial products embedded with ESP32 moduels.

Recently I purchased one FireBeetle ESP32-E IoT board from Dfrobot.com.

Figure 1. My Dfrobot ESP32 board

It supports WiFi and Bluetooth dual-mode communication and features a small size, ultra-low power consumption, onboard charging circuit, and easy-to-use interface, which can be conveniently used for smart home IoT, industrial IOT applications, wearable devices and so on. You can easily create your own IoT smart home system when connecting it with an IoT platform like IFTTT.

ESP32 supports Arduino programming, ESP-IDF programming and also microPython programming.

In this post, I will share about the environment setup on your computer. I have tried Arduino on my Ubuntu20.04 and ESP-IDP + Visual Studio Code on windows. Both setup are not complicated, and depends on your favor, you can choose one to start. I talked to few users, most of their impression is: Arduino would be quick for prototype, and ESP-IDF is more product level positioned.

If you are a starter, I recommend you start from Arduino, it is extremely convenient and easy to get your hands on.

ESP32+Arduino

Arduino, in case you do not know, is an open-source electronics platform based on easy-to-use hardware and software. It starts from Atmel MCU platform, and when it is getting more popular, there are more and more chips are supported, also includes ESP32.

Figure 2. Arduino IDE

Arduino has one IDE, which is cross platform, you can download and install different OS version from its website: https://www.arduino.cc.

Then:

1, install Arduino IDE on your computer

2, install the ESP32 support using Boards manager.

Open the Preference, in the green rectange “Additional Boards Manager URLs”, paste the json file link. There are two options:

  • One from Espressif official (https://docs.espressif.com/projects/arduino-esp32/en/latest/installing.html#installing-using-boards-manager): https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json

  • One from Dfrobot (https://wiki.dfrobot.com/FireBeetle_Board_ESP32_E_SKU_DFR0654): http://download.dfrobot.top/FireBeetle/package_DFRobot_index.json

I tried both for experiment, compared to Dfrobot json, Espressif one has more boards supported and very rich examples (also available on its git). First one is really recommended.

Copy&paste the url, click OK.

Figure 3. Add board json in Arduino preference

3, Update board, open Tools->Board->Board Manager, search the keyword “esp32”, find the item and click “install”

Figure 4. Install the board in board manager

It will download three files, total size is few hundreds of Mb, wait until it is completed.

4, you can find ESP32 boards side by side with Arduino AVR ones, and ready for use:

Figure 5. Various ESP32 board supported

5, additionally,  there are lots of examples available for ESP32 after the board is installed.

Figure 6. always good to start with some examples

6, start from an example or new project, select the board (Firebeetle-ESP32), coding, compile and upload, that’s all.

ESP32 + ESP-IDF + VS Code

ESP-IDF is Espressif’s official IoT Development Framework for the ESP32, ESP32-S and ESP32-C series of SoCs. It provides a self-sufficient SDK for any generic application development on these platforms, using programming languages such as C and C++.

Installing esp-idf has several solutions: 1, git clone from github: https://github.com/espressif/esp-idf 2, option of windows installer, https://dl.espressif.com/dl/esp-idf/?idf=4.4 3, use Espressif plugin in VS code.

The first or two option works fine, you can  find how-to in the links, once installed, you can use idf shell with shell commands to do config, build and flash. But I would strongly recommend the vs code way, it is so amazing to use.

Let me share the setup steps:

1, install the vs code, it is also cross OS platform

2, in the market place, search “Espressif IDF”, click install

3, on the left menu bar, find the ESP-IDF icon, and you can see the installation guide.

Figure 7. install Espressif IDF plugin in VS code

4, choose the “EXPRESS”, and check the options below, normally, you do not need to change anything.

If you feel github download is slow, you can choose “Espressif” in “Select download server”.

Click install.

Wait till all the components are ready.

Figure 8. ESP-IDF extension Installation Options

5, now look at the interface

In the bottom menu, there are few handy buttons for daily work:

  • Serial Port Selection
  • Chip Selection
  • Menu-config
  • Clean
  • Build
  • Flash
  • Serial Monitor
  • Build, Flash and Monitor

Figure 9. Handy IDE function in VS code with ESP-IDF extension

6, You can code now. Examples are good starting point, their location is /Users/<username>/esp/examples/esp-idf/examples, pick one and play with it. You can also find the latest examples in the github: https://github.com/espressif/esp-idf

That’s all for this post. If you follow it, you can setup your own ESP32 IDE with Arduino or VS code.
