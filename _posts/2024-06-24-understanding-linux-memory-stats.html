In Linux, the terms "free" and "available" memory have specific meanings when you check system memory usage, often using commands like free or top.

Free Memory

"Free" memory refers to the amount of physical RAM that is currently not used at all by the system. This is memory that has no data stored in it, not even cached data. It's essentially memory that is entirely unused.

Available Memory

"Available" memory is a more comprehensive metric that includes:

  • Free Memory: The truly unused memory.

  • Cached Memory: Memory used by the system to cache disk data for faster access, which can be quickly repurposed for other tasks if needed.

  • Buffers: Memory used by the system for temporary storage of data being moved in and out of devices.

  • Inactive Memory: Portions of memory that were recently used by processes but are not actively being used at the moment and can be reclaimed if necessary.

In essence, "available" memory represents the amount of memory that can be used for new processes without needing to swap out other data or increase the overall system load.

Checking Memory in Linux

You can check both "free" and "available" memory using the free command:

$ free -h

Output:

total used free shared buff/cache available Mem: 7.7G 2.3G 300M 100M 5.0G 5.1G Swap: 2.0G 0.0G 2.0G

  • total: Total installed memory.

  • used: Memory currently used by processes.

  • free: Truly free memory.

  • shared: Memory used (mostly) by tmpfs.

  • buff/cache: Memory used by the kernel buffers and page cache.

  • available: An estimate of how much memory is available for starting new applications, without swapping.

Understanding the Differences

  • Free Memory: It's the truly idle memory, not being used or reserved by any processes or the OS.

  • Available Memory: It's the memory readily available for new processes, considering the ability of the OS to reclaim memory from buffers, cache, and inactive pages.

Using the free command with the -h flag provides a human-readable format, making it easier to understand the memory usage statistics.
