I want to add website register number in home page. This requires me to do some customization on wordpress themes, I explored ways of doing it. Share to you if you have similar needs.

Preparation:
change theme folder file permission from 755 to 777, So that Wordpress program can write files.

$ chmod -R 777 html/wp-content/themes

Upper step is optional, if you have correct file permission for your wordpress, it is not required.

Preliminary Customization:
Wordpress already provides some general customization 

1, change site identity: Give a site your name

2, colors

3, header images

4, homepage settings

With these, we can already make site with our own features

Below picture shows more options in the menu

for our selected theme, eg. my “Markup” theme allows us to do further.

in “Markup Theme options”, we can

1, change footer setting, we can update year in "Copyright All Rights Reserved 2022”

2, single page setting, here we can change page layout

There are more things that we can customize on our own theme. Try the features in below pic:

Advanced customization:

Further, if upper theme options cannot fulfill our requirement, we can change the theme raw file.

Eg. Replace “Powered by XXX” with our site identity string.

We can do this via: Appearance - Theme File Editor

Be cautious, changes on the theme raw file may break our page, make a backup before we do it. And you need to have basic knowledge of html, css.

One more thing, if we upgrade theme to new version, our changes will get lost. Remember to recover those changes after we do upgrade.

My requirement is to change the theme.footer, and add my Chinese site register number in it.

find the theme.footer in edit mode:

change below:

<?php
   /* translators: 1: Theme name, 2: Theme author. */
   printf( esc_html__( 'Theme: %1$s by %2$s.', 'markup' ), 'Markup', '<a href="https://www.templatesell.com/">Template Sell</a>' );
?>

to text like:

<a href="http://www.beian.miit.gov.cn" rel="nofollow”>{your register no}</a>

if you previously change the file permission, do change back the folder permission to 755 

$ chmod -R 755 html/wp-content/themes

Customization Effect
Finally, try this out on your new homepage, it already show the register number in footer.
