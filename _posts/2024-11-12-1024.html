Background:

umap-learn

• Purpose: umap-learn is a Python library for dimensionality reduction, commonly used to simplify high-dimensional data for visualization and clustering.

• Key Concept: UMAP stands for Uniform Manifold Approximation and Projection, which is an algorithm designed to reduce complex, high-dimensional data into a lower-dimensional space (often 2D or 3D). It preserves the underlying structure of the data, making it useful for visualizing relationships or clusters.

• Common Applications:

• Visualizing high-dimensional datasets in 2D or 3D.

• Preprocessing for clustering, classification, or other machine learning tasks where lower-dimensional data is useful.

The umap.UMAP(n_neighbors=n) parameter in the UMAP (Uniform Manifold Approximation and Projection) library sets the number of neighboring data points to consider when learning the structure of the data manifold. Here’s a breakdown of what n_neighbors does and its effect:

What n_neighbors Does

• n_neighbors defines the local neighborhood size for each data point in the high-dimensional space.

• UMAP tries to preserve the structure of the data by examining the local relationships between points, essentially by connecting each point to its nearest neighbors.

• By setting n_neighbors=n, you specify the number of nearest points each data point should consider when UMAP constructs a lower-dimensional representation.

How n_neighbors Affects the Output

• Smaller n_neighbors values (e.g., n_neighbors=5) make UMAP focus more on capturing local structure, which may reveal finer details or clusters within the data. This is useful if you want to emphasize small clusters or local relationships.

• Larger n_neighbors values (e.g., n_neighbors=50) encourage UMAP to consider broader patterns and more global structure, which can make the embedding more cohesive and better at representing global clusters.


import umap
import numpy as np

#Example data with 100 points in 10D space
data = np.random.rand(100, 10)

#Initialize UMAP with a specified n_neighbors value
n = 15
reducer = umap.UMAP(n_neighbors=n)
#Fit and transform the data to 2D
embedding = reducer.fit_transform(data)

In this example:

• Setting n_neighbors=15 means each data point will consider its 15 nearest neighbors in the high-dimensional space.

• UMAP will then project these points to a 2D space, preserving local neighborhood relationships based on the specified number of neighbors.

The n_neighbors parameter is one of the main ways to control the focus of the UMAP embedding.

Now, you can visualize the data in 2D

import matplotlib.pyplot as plt
plt.figure(figsize=(8, 6))
plt.scatter(embedding[:, 0], embedding[:, 1], s=10, c='blue', alpha=0.7)
plt.title("2D UMAP Projection")
plt.xlabel("UMAP Dimension 1")
plt.ylabel("UMAP Dimension 2")
plt.show()
