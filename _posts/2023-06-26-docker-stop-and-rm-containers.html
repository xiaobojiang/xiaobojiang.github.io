To stop all running Docker containers and remove them, you can use the following command:

docker stop $(docker ps -aq) && docker rm $(docker ps -aq)

The -q option in the docker command is used to display only the container IDs. The -a option is used to show all containers (default shows just running).

So, the -aq option is used to display all container IDs (running and stopped).

This command stops all running containers and removes them.

I hope this helps!
