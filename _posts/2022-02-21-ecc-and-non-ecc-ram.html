It is orginated from my personal story.

I have obtained a HP Z820 server from my colleague, and it equips with 4*4G DDR3 RAM.

I was running some pytorch program, then the RAM ran out. I thought this is simple, I purchased 4 * 8G Samsung DDR3 1600MHz. The replacement is easy, but the sad story begins, the server cannot start, and cpu fan goes crazy after the boot, and of course the booting fails.

The root cause is related to the post topic: memory ECC feature.

What is ECC

Error Correction Code (ECC) memory, it is a kind of RAM that often found in workstations and servers. It has ability to automatically detect and correct memory errors, thus fighting data corruption. It is also supposed to lead to less crashes of a server workstation over non-ecc memory.

ECC vs non-ECC

They have some difference:

1, Reliability

Though ECC memory is theoretically more stable and reliable compared with traditional RAM. The failure rate of non-ECC memory is higher than that of ECC in the past 3 years.

2, Performance

As claimed by many memory manufacturers, ECC memory is 2% slower than standard RAM because the additional time it takes for the system to check for any memory errors.

3, Price

Due to the additional memory chip and the inherently more complicated nature of ECC RAM, it charges higher than non-ECC memory. The amount varies according to the size of the memory stick. My real experience, from the same vendor, the same RAM volume, ECC charges 80% more.

How to differentiate?

As a non professional in RAM, I cannot tell the ECC or non-ECC from the label. However, please do not worry, there is one easy way to differentiate: number of chips on the RAM stick.

Non-ecc type RAM has 8 chips, and ECC type shall have 9 chips. ECC’s additional chip is used for error detection and correction among the other eight memory chips.

Can they use together?

No

Can they replace each other?

Depends. From that I searched on the internet, for certain mother board, in the BIOS setting, the ECC feature can be turned off. For my Z820, unfortunately, these is no such option, which means ECC seems to be a must.

Story end

In order to upgrade the server, I returned the non-ECC RAM, spend 80% more to get 4*8G ECC RAM.

So, please also pay attention when you face the similar scenario..

And, consulting the seller is always best option before you take the real action.
