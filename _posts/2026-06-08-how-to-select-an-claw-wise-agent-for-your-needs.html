For resource limited device, running openclaw would require too many RAM and storage. So I need an alternative, with below few things in mind: small footprint with limit RAM and storage request, has good track of github stars and under active maintenance, support skills, support running in container way.

Below is a weighted scoring table focused on my request:

Scoring weights of my certain application scenario, for your reference.

  • Small footprint: 30 pts — smaller RAM/storage gets higher score.

  • GitHub stars: 20 pts — higher current stars get higher score.

  • Skills support: 30 pts — hard requirement; downloadable + usable skills score highest.

  • Container + persistence: 20 pts — can fully run in container and preserve state via mounted folder/volume.

Recommended shortlist for my application criteria
