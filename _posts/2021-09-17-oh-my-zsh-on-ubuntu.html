"Zsh is a shell designed for interactive use, although it is also a powerful scripting language."

I first started to use zsh is when I have my first Macbook pro. previously I am using bash on linux, and one colleague recommend me with Zsh, "it is cool!"

Zsh offers big freedom for customization, installing plugins, and so many cool themes.

Later when I have one desktop server installed with Ubuntu, I just think, why not Zsh.

Installing on MacOS is rather simple:

$ brew install zsh zsh-completions

Installation on Ubuntu is also simple

$ sudo apt install zsh

for Ubuntu, you need to execute following command to make the zsh as default shell:

$ sudo chsh -s $(which zsh)

exit current terminal and enter again, you will find zsh style comes.

You may be asked by Zsh to do first-time configuration, follow the guide and select 2 to pupulate the zshrc file with configuration recommended.

Now that zsh is installed we will move on to Oh My Zsh a framework that allows you to customize your Shell with themes and plugins.

Oh My Zsh is an open source, community-driven framework for managing your Zsh configuration.

It provide you with a simple line of command to download and install oh-my-zsh.

Then, let's install few basic productive plugin in zsh: zsh-autosuggestions and zsh-syntax-highlighting

$ git clone https://github.com/zsh-users/zsh-autosuggestions.git $ZSH_CUSTOM/plugins/zsh-autosuggestions
$ git clone https://github.com/zsh-users/zsh-syntax-highlighting.git $ZSH_CUSTOM/plugins/zsh-syntax-highlighting

Edit the zshrc file to add into plugin list

$ vim ~/.zshrc

# Which plugins would you like to load?
# Standard plugins can be found in $ZSH/plugins/
# Custom plugins may be added to $ZSH_CUSTOM/plugins/
# Example format: plugins=(rails git textmate ruby lighthouse)
# Add wisely, as too many plugins slow down shell startup.
plugins=(git zsh-autosuggestions zsh-syntax-highlighting)

try it out

One more thing, you can select themes for you zsh.

Personally, I like "dst".

Also, set it in the zshrc file

# Set name of the theme to load --- if set to "random", it will
# load a random theme each time oh-my-zsh is loaded, in which case,
# to know which specific one was loaded, run: echo $RANDOM_THEME
# See https://github.com/ohmyzsh/ohmyzsh/wiki/Themes
ZSH_THEME="dst"

You can set the value to "random", it will load random theme, you may find your favorite by your daily use

Zsh also renders very nice github information when you are inside a git folder.
