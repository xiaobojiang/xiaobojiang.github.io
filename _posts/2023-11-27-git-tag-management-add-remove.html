Create tag

Creating an annotated tag in Git is simple. The easiest way is to specify -a when you run the tag command:

$ git tag -a v1.4 -m "my version 1.4”

Push tag

To push a single tag:

$ git push origin <tag_name>

Remove Tag

To remove a remote tag in Git using the command line, you can use the git push command with the --delete option. Here's the general syntax:

$ git push <remote_name> --delete <tag_name>

Replace <remote_name> with the name of your remote repository (usually "origin" by default), and <tag_name> with the name of the tag you want to delete.

For example, if you want to delete a tag named "mytag" on the "origin" remote, you would run:

$ git push origin --delete mytag

After executing this command, the tag will be removed from the remote repository. Note that this operation only removes the tag from the remote repository; it doesn't delete the tag from your local repository. If you also want to delete the local tag, you can use the following command:

$ git tag -d mytag

This command deletes the local tag named "mytag."

Remember to use these commands with caution, especially when deleting remote tags, as they can't be easily recovered once deleted. Always make sure you have a backup or confirmation before removing tags, especially if they are shared and used by others.
