Counter is a Python import statement that brings in the Counter class from the collections module.

What is Counter?

Counter is a specialized dictionary subclass used for counting hashable objects. It counts the occurrences of each element in an iterable and stores them as dictionary key-value pairs, where the keys are the elements and the values are their counts.

Example Usage:

  1. Counting Elements in a List

from collections import Counter
data = ['apple', 'banana', 'apple', 'orange', 'banana', 'apple']
counter = Counter(data)
print(counter)
Output: Counter({'apple': 3, 'banana': 2, 'orange': 1})

  1. Counting Characters in a String

text = "hello world"
char_count = Counter(text)
print(char_count)
Output: Counter({'l': 3, 'o': 2, 'h': 1, 'e': 1, ' ': 1, 'w': 1, 'r': 1, 'd': 1})

  1. Finding the Most Common Elements

most_common = counter.most_common(2)  # Get top 2 most common elements
print(most_common)
Output: [('apple', 3), ('banana', 2)]

  1. Using Counter for Arithmetic Operations


counter1 = Counter({'a': 3, 'b': 2, 'c': 1})
counter2 = Counter({'a': 1, 'b': 2, 'c': 3})
print(counter1 + counter2)  # Adds counts together
Output: Counter({'a': 4, 'b': 4, 'c': 4})
print(counter1 - counter2)  # Subtracts counts, removes zero or negative counts
Output: Counter({'a': 2})
print(counter1 & counter2)  # Intersection: min of corresponding counts
Output: Counter({'a': 1, 'b': 2, 'c': 1})
print(counter1 | counter2)  # Union: max of corresponding counts
Output: Counter({'a': 3, 'b': 2, 'c': 3})

Summary

• Counter is useful for counting occurrences of elements in a list, string, or any iterable.

• It provides quick ways to get the most common elements.

• Supports arithmetic operations like addition, subtraction, intersection, and union.
