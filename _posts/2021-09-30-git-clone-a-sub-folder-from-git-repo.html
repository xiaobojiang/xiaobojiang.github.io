Sometimes, you want to clone a subfolder in the big repo. Pulling the whole may takes too long time, and other files are actually not requried.

There is a new feature in Git to support this (from git 2.19)

Below are the sample code, it can help you download folder "examples" only in the repo: https://github.com/espressif/esp-idf

$ git clone \
  --depth 1  \
  --filter=blob:none  \
  --sparse \
  https://github.com/espressif/esp-idf
  
$ cd esp-idf
$ git sparse-checkout set examples
