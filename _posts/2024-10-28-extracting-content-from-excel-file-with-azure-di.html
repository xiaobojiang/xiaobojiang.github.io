It is a common practice to extract content from different file formats, so it can be further processed as corpus for AI.

Azure AI Document Intelligence (formerly known as Azure Form Recognizer) is machine-learning based service that extracts texts (including handwriting), tables, document structures (e.g., titles, section headings, etc.) and key-value-pairs from digital or scanned PDFs, images, Office and HTML files.

Document Intelligence supports PDF, JPEG/JPG, PNG, BMP, TIFF, HEIF, DOCX, XLSX, PPTX and HTML.

Here, to simplify the code, we leverage langchain wrapped loader class AzureAIDocumentIntelligenceLoader for the task. It can output Langchain documents object, which can be easily connected with other langchain components for next step, eg splitter for chunking.

I tried, Azure DI can support excel format of xlsx, and xls, however xlsm (with macro) extension is out of scope.

Prerequisite:

-> Azure DI service, you shall have <endpoint> and <key> for next step experiment.

Dive in a little bit how loader part is done.

1, Load the document.

With provisioned api_endpoint, api_key, and file_path, AzureAIDocumentIntelligenceLoader can do the job.

The api model is a interesting part, Azure DI do provide a lot of customer models for specific documents (eg, tax, invoice), check more here: https://learn.microsoft.com/en-us/azure/ai-services/document-intelligence/how-to-guides/use-sdk-rest-api?view=doc-intel-4.0.0&tabs=windows&pivots=programming-language-csharp

    try:
        loader = AzureAIDocumentIntelligenceLoader(
            api_endpoint=endpoint, api_key=key, file_path=file_path, api_model="prebuilt-layout",
            analysis_features=["keyValuePairs"]
        )
        documents = loader.load()
    except Exception as e:
        print(f"Error loading document, filename: {file_path}, exception: {e}")
        return parsed_documents

2, split into sheets

looking into the content in the documents object.

all sheets are in one document, splitted by "<!-- PageBreak -->".

and sheet name is first section in each part, wrapped with # sheetname \n\n

so make regex pattern r"# (.+?)\n\n", and find the first would work very well.

    page_break = "<!-- PageBreak -->"
    pattern = r"# (.+?)\n\n"
    print(f"analyzing document, filename: {file_path}")
    for document in documents:
        document_json = json.loads(document.json())
        file_name = os.path.basename(file_path)
        content = document_json.get('page_content', None)

        if content:

            sheets = content.split(page_break)

            for sheet in sheets:
                # Regular expression pattern
                # Search for the pattern in the text
                match = re.search(pattern, sheet, re.DOTALL)

                if match:
                    sheet_name = match.group(1)
                else:
                    sheet_name = "No name found"

                sheet_content = sheet
                parsed_document = ParsedDocument(
                    file_name=file_name,
                    sheet_name=sheet_name,
                    sheet_content=sheet_content
                )
                parsed_documents.append(parsed_document)
    return parsed_documents
