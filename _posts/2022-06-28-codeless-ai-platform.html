no-code AI platforms offer automated programming tools that either generate new code or customize pre-existing code according to user input — generally requiring only access to a web browser and the training data for the ML task at hand

1, Teachable machine (Google & Lobe.ai)

https://teachablemachine.withgoogle.com/

Has three main categories of features:

  • Image classification
  • Audio classification
  • Posture project

Take image project as example, user can upload pictures or took by webcams into different classes, and generation model. This model then can be used for classification of new pictures

User can export model to tensorflow js, tensorflow or tensorflow lite

2, Power platform (Microsoft)

https://powerplatform.microsoft.com/en-us/

It is BI system, can connect to all Microsoft data resources.

Microsoft Power Platform—analyze data, build solutions, automate processes, and create virtual agents.

3, AI builder (Microsoft)

https://docs.microsoft.com/en-us/power-apps/use-ai-builder

AI Builder is a new Power Platform capability for teams with the business expertise to easily automate processes and predict outcomes to improve business performance. AI Builder is a turnkey solution that brings the power of Microsoft AI through a point-and-click experience and is directly integrated into Power Apps and Power Automate.

4, AWS SageMaker Canvas

https://aws.amazon.com/sagemaker/canvas/

Amazon SageMaker Canvas expands access to machine learning (ML) by providing business analysts with a visual point-and-click interface that allows them to generate accurate ML predictions on their own — without requiring any machine learning experience or having to write a single line of code.
