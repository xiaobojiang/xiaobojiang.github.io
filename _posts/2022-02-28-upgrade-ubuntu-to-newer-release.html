If you own computer with older Ubuntu release, eg 14.04, 18.04.

You found that certain versions are phasing out, and you also want to try new features in new release.

Here, follow the steps and upgrade Ubuntu.

1, check current version

2, update your current system 

$ sudo apt update  
$ sudo apt upgrade

3, decide shall we want to upgrade to latest LTS (long term support) version or non-LTS development version

Current LTS version is 20.04 Non-LTS is 21.10

By default system will upgrade to LTS version.

if we want non-LTS version, go to and edit file: 

/etc/update-manager/release-upgrades

Change prompt from “lts” to “normal”.


4, start upgrade. 

$ sudo do-release-upgrade

It will download many packages.

Resolve some package changes during upgrade, normally just follow the suggestion.


5, reboot and here you go.
