The round() function in Python is a built-in function used to round a number to a specified number of decimal places. Here’s an explanation of its functionality and syntax:

round(number, ndigits)

Parameters

  1. number (required):

The number to be rounded. It can be an integer or a floating-point number.

  1. ndigits (optional):

The number of decimal places to round to. It can be:

• Positive: Specifies the number of digits after the decimal point.

• Zero or None: Rounds to the nearest integer.

• Negative: Rounds to a position to the left of the decimal point.

Return Value

• Returns the rounded value of number.

• The return type matches the input type:

• If number is an integer and ndigits is not specified, it returns an integer.

• Otherwise, it returns a float.

Rounding to Decimal Places

Round to 2 decimal places

print(round(3.14159, 2)) # Output: 3.14

Round to 0 decimal places

print(round(3.75)) # Output: 4

Behavior on Tie (Bankers’ Rounding)

When the number is exactly halfway between two possible rounded values (e.g., 2.5), Python uses “bankers’ rounding”:

• It rounds to the nearest even number.

print(round(2.5)) # Output: 2 (nearest even)

print(round(3.5)) # Output: 4 (nearest even)

This behavior is based on minimizing rounding errors in repeated calculations.

try below two approaches:

Method 1: Add a Small Offset

Adding a tiny offset ensures the rounding favors the higher number:

result = round(7.05 + 1e-9, 1)

print(result) # Output: 7.1

Method 2: Use the decimal module

def generate_decimal_precision(precision):
    return Decimal(f"1e-{precision}")
def round_float(value: str, precision=2):
    """
    Round the float value with the given precision.
    """
    try:
        precisioner = generate_decimal_precision(precision)
        result = Decimal(value).quantize(precisioner, rounding=ROUND_HALF_UP)
        return  str(result)
    except InvalidOperation:
        pass
    return value
if name == 'main':

    print(round_float("7.05", 1))
    print(round_float("14.05", 1))
    print(round_float("cm"))
