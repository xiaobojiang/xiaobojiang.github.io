I want to fetch swagger from /dev/docs, and with that, openapi.json is also fetch from /dev/openapi.json

To set the root URL for Swagger UI in FastAPI, you can use the openapi_url parameter in the FastAPI constructor1. Here’s an example:

fromfastapiimportFastAPI

app = FastAPI(

    openapi_url="/dev/openapi.json",

    docs_url="/dev/docs",

    redoc_url=None,

)

In FastAPI, the openapi_url, docs_url, and redoc_url parameters are used to configure the URLs for the OpenAPI schema, Swagger UI, and ReDoc UI, respectively 1. Here’s what each parameter does:

  • openapi_url: This parameter specifies the URL for the OpenAPI schema. By default, it is set to /openapi.json. You can customize it by setting this parameter to a different URL 2.

  • docs_url: This parameter specifies the URL for the Swagger UI. By default, it is set to /docs. You can customize it by setting this parameter to a different URL 3.

  • redoc_url: This parameter specifies the URL for the ReDoc UI. By default, it is set to /redoc. You can customize it by setting this parameter to a different URL 1.

ReDoc is an alternative UI for OpenAPI documentation. It provides a clean and modern interface for exploring and testing APIs. ReDoc is included in FastAPI by default, and you can enable it by setting the redoc_url parameter in the FastAPI constructor

In this example, the root URL for the app is still /. The openapi_url parameter specifies the URL for the OpenAPI schema, and the docs_url parameter specifies the URL for the Swagger UI. The redoc_url parameter is set to None to disable ReDoc.

importuvicorn

app = FastAPI(

openapi_url="/dev/openapi.json",

docs_url="/dev/docs",
