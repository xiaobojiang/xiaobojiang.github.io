It is just because you clone other’s repo into your repo

when you use git add to add these files, it will show this error： “fatal: in unpopulated submodule XXX”

Correct way is to add this a submodule

$ git submodule add git://XZXX.git

then it is fine to add files and push

$ git add -A

$ git commit -m ’note’

$ git push
