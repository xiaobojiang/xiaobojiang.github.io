We do not need to open the shield of RPI, and find the model on the PCB board, here, I would like to share the way to find it via command line.

To have the terminal display the RPI model number:

1, open the terminal (do it via local tty or remote ssh)

2, input in the command line:

$ cat /sys/firmware/devicetree/base/model && echo ''

3, read the Raspberry Pi model number:

Cool, this is my powerful RPI4.

An alternative of upper command is 

$ cat /proc/cpuinfo

you can find the model info at the end of this command line output

That's it. This can help you know the model if you manages multiple RPI machines.
